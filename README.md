# IrinaArevalo.github.io

Personal academic webpage of **Irina Arévalo**, Assistant Professor of Applied Mathematics at **Universidad Politécnica de Madrid**. The site includes professional information such as positions, research interests, publications, teaching, students, accreditations, and teaching resources.  

## Live website

Visit the website here: **[irinaarevalo.github.io](https://irinaarevalo.github.io/)**.  

## About

This repository contains the source code for my personal academic website hosted with **GitHub Pages**. The site is based on the **Minimal** theme and has been adapted to present my academic profile, research, teaching, and publications. 

## Website sections

The current website includes the following sections:  

- Positions
- Research
- Publications
- Teaching
- Students
- Accreditations
- Teaching Resources

## Updating the site

To update the webpage:

1. Edit `index.html` to modify content and sections.
2. Update styles in `stylesheets/`.
3. Add or replace assets in `images/` and `files/`.
4. Push changes to the repository to publish them through GitHub Pages.
